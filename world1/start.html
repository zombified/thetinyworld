The Tiny World

A text adventure by Joel Kleier

Created between April 20th and 22nd, 2012

Welcome! Here's a few things you should know to get started:

  • If you click in the grey area at the bottom of the screen, you can start typing! This is the command area -- hitting the enter key will execute the command you typed in!
  • At any point you can type in help in the command field, then hit the enter key, to get a list of possible commands, and other, hopefully helpful, information.
  • Have fun! Remember, a part of playing this text adventure is figuring out what you need to type in to do something. It's part of puzzle!
  • This game was created from scratch in less than 48 consecutive hours as part of the Ludum Dare. If there is poor spelling and grammar, loads of bugs, or just a bad game -- that's par for the course!

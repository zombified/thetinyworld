Welcome! Here's a few things you should know:
  • If you click in the grey area at the bottom of the screen, you can start typing! This is the command area -- hitting the enter key will execute the command you typed in!
  • At any point you can type in ? (that is, just a question mark) in the entry field, then hit the enter key, to get a list of possible commands
  • Movement is controlled by the following commands: n, e, s, w -- which represent the 4 cardinal directions
  • The a command attacks whatever is in front of you
  • The i command lists the contents of your inventory
  • The l command describes what you see in the direction you are facing.
Move north to start your game, oh, and have fun!
